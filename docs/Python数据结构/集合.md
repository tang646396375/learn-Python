# 集合

特点 : 无序不重复

```
list1 = [1,3,5,6,5,3,1]
set1 = set(list1)   # (1,3,5,6)
```

![1547815379419.png](image/1547815379419.png)

* 集合不能包含重复项
* 集合不能被切片也不能被索引
* 集合运算+增删
